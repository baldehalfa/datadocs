# dbdocs setup and commands

### Install dbdocs
```sudo npm install -g dbdocs ```

### check version
```dbdocs```

### Login to dbdocs
```dbdocs login```

### Generate dbdocs view
```dbdocs build <path to your dbml file>/<file name>.dbml```

### Remove project
```dbdocs remove <project name>```
